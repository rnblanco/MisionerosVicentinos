Documentación del proyecto

Este proyecto fue desarrollado para Misioneros Vicentinos de América Central y su versión original puede encontrarse aquí.

Sin embargo, esta es una versión desplegada con el objetivo de compartir mis proyectos como desarrollador web, así que este sitio y su contenido no representan a Misioneros Vicentinos de América Central ni derivados.

El proyecto consiste en dos vistas, la de Usuario y Administrador.

Usuario:

Se planteó que los usuarios no requieran inicio de sesión para observar el contenido de la organización. De esa manera, los usuarios pueden acceder a un carrusel de imágenes, diferentes artículos de opinión y la posibilidad de descargarlos como archivos pdf, una galería de santos con sus historias y una galería de todas las imágenes que pertenecen al proyecto. Es posible acceder aquí.

Administrador:

La vista de usuarios administradores si require de inicio de sesión y para hacer modificaciones y probar todas las funcionalidades de la aplicación se proveen como usuarios de prueba los siguientes:

  • Usuario: Super Admin | Contraseña: superadmin
  • Usuario: Admin | Contraseña: admin
Nota:

Se recomienda no eliminar el usuario BLANCO, pues al hacerlo, se eliminarán todas las publicaciones del sitio con él.

Nota 2:

Se recomienda crear un usuario de prueba de tipo ESCRITOR desde el panel de administrador o súper administrador para hacer las pruebas pertinentes.

Para entrar al panel de administrador, presiona aquí.

Sin más que agregar, espero que lo disfrutes :)
